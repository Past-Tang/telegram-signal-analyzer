## 概述

Telegram 消息分析与交易信号监控工具集。监控 TG 频道/群组消息，提取交易信号，支持自动化交易执行。包含消息历史导出、频道监控、交易信号解析等功能。

## 功能模块

- **消息监控** -- 实时监控 Telegram 频道交易信号
- **历史导出** -- 导出频道/话题的历史消息
- **信号解析** -- 解析交易信号并生成结构化数据
- **自动交易** -- 对接 Gate.io API 执行交易
